# Gateway (`@breeze/gateway`)

WebSocket gateway that handles all real-time streaming between clients and sandboxes (Client ↔ Gateway ↔ Sandbox).

- Spec: `SPEC.md`
- Docs: https://docs.breeze.engineer/introduction/core-concepts

## Development

- From repo root: `pnpm dev:gateway`
- Tests: `pnpm --filter @breeze/gateway test`
